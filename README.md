## Description

My Hunter is a simple 2D game developed in C using the CSFML library.  
The game features sprite rendering, user input handling, collision detection, scoring, and a continuous game loop.

## Project Goals

- Learn to use CSFML for 2D graphics
- Manage game loops and timing
- Handle user input and events
- Display and animate sprites
- Implement collision detection and scoring
- Structure a graphical C program

## Technologies Used

- Language: C
- CSFML (C Graphical Library)
- Makefile

## Features

- Window creation and configuration
- Sprite loading and rendering
- Keyboard and mouse input handling
- Basic game mechanics and object movement
- Error handling

## Compilation

Before compiling, make sure CSFML is properly installed:

make

- ./my_hunnter
- ./my_hunter -h (for usage parameters)
